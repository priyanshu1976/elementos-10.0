# Auction Website – Style Guide

## Theme
Primary: #0D1117  
Secondary: #161B22  
Accent: #58A6FF  
Danger: #FF4D4F  
Success: #22C55E  
Text: #E6EDF3  
Muted Text: #8B949E

---

## Fonts
Headings: Poppins / Inter  
Body: Inter / Roboto  
Numbers: JetBrains Mono

---

## UI Style
- Glassmorphism cards
- Rounded corners (12px)
- Soft shadows
- Neon accent highlights
- Subtle gradients
- Minimal icons

---

## Components

### Buttons
- Glow hover
- Accent borders
- Disabled gray state

### Cards
- Dark glass background
- Blur effect
- Border glow on hover

### Timer
- Circular progress ring
- Color shift:
  - Green → Yellow → Red

### Tables
- Thin borders
- Hover highlight
- Sticky headers

---

## Animations
- Fade in dashboard
- Slide bid updates
- Pulse highest bid
- Shake on invalid bid

---

## Layout
- Sidebar for admin
- Top navbar for teams
- Grid cards
- Centered auction area

---

## UX Rules
- Always show timer
- Always show current item
- Always show money
- Prevent double clicks
- Confirmation before bid
